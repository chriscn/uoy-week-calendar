University of York's Week Numbers

This page provides an iCalendar (RFC 5545) format of the University of York's week numbers, according to the term dates.

It is compatible with most calendar software, the program provides three different calendars depending on your circumstance. Simply copy one of the following URL to your Calendar software:

  • Undergraduate - https://chriscn.github.io/uoy-week-calendar/calendar/undergraduate.ics
  • Postgraduate - https://chriscn.github.io/uoy-week-calendar/calendar/postgraduate.ics
  • Staff - https://chriscn.github.io/uoy-week-calendar/calendar/staff.ics

The program should support up to the academic year 2029/2030; and is easily updatable past that. However, it will only generate the current academic year's week numbers.

Format

From September 2023; each week can be described as one of the following types:

  • f/rf: Freshers / Refreshers week - no teaching
  • 1 - 11: Teaching and learning week
  • c: Consolidation
  • rv: Revision and essay writing weeks - free from contact time and scheduled assessment
  • ra: Revision and assessment weeks. Departments will set examinations and end of semester assessment deadlines
  • m: Marking - free from teaching
  • o: Research, admin duties, PGR/PGT supervision, summer resits, teaching prep, student recruitment activities, vacation

Application Programming Interface (API)

In lieu of a dynamic API, there exists a file called dates.json. This file contains a list of dates that are Mondays, and the corresponding week number in a dictionary. One can then use this file locally.
